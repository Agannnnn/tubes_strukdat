Aplikasi pemutar musik sederhana yang memiliki fitur-fitur seperti berikut:
1. menyimpan lagu,
2. mengakses lagu (putar),
3. menghapus lagu,
4. memperbarui lagu,
5. membuat playlist,
6. menghapus playlist,
7. memperbarui playlist (tambah & hapus lagu),
8. membuat antrian lagu,
9. menghapus antrian lagu,
10. memperbarui antrian lagu (tambah & hapus lagu),
11. memiliki dua jenis pengguna (admin & user),
12. memutar lagu (kalau sempat).
